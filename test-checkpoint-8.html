Checkpoint 8: Drawing Tools and Annotation Manipulation Test

This test verifies that drawing tools and annotation manipulation functionality is working correctly.

Test Controls

Manual Testing Instructions:
  • Drawing: Click and drag to create bounding boxes
  • Selection: Click on annotations to select them
  • Resizing: Drag the corner handles of selected annotations
  • Moving: Drag selected annotations to move them
  • Context Menu: Right-click on annotations for options
  • Double-click: Double-click annotations to edit class/state
  • Keyboard: Delete key to remove selected annotations

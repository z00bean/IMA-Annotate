# Table 1 — FHWA Motor Vehicle Classes (Scheme F)
Use this table for motor-vehicle traffic counts, TMC, pavement, safety, and exposure analysis.

| Annotated Vehicle | FHWA Class | FHWA Official Class Name                     | FHWA Class Definition |
|------------------|------------|---------------------------------------------|----------------------|
| Motorcycle       | Class 1    | Motorcycles                                 | Two- or three-wheeled motor vehicles |
| Car              | Class 2    | Passenger Cars                              | 2-axle, 4-tire passenger cars, sedans |
| Van              | Class 2    | Passenger Cars                              | 2-axle, 4-tire vans and minivans |
| Pickup truck     | Class 2    | Passenger Cars                              | 2-axle, 4-tire pickups and SUVs |
| Bus              | Class 4    | Buses                                       | All 2-axle or 3-axle buses |
| Medium truck     | Class 5–6  | Single-Unit Trucks                          | Single-unit trucks with 2 or 3+ axles (e.g., box trucks, garbage trucks, cement mixers) |
| Semi (tractor)   | Class 7–13 | Combination Trucks                          | Multi-axle tractor-trailer combinations |
| Truck trailer    | Class 7–13 | Combination Trucks (Trailer Component)     | Trailer unit of a combination truck (counted as part of the tractor-trailer system) |
| Car trailer      | Class 7–13 | Combination Vehicles                       | Trailer towed by a passenger vehicle or pickup (counted as a combination vehicle) |

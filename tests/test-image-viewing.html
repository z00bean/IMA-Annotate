Manual Testing Instructions

  1. Open the main application: index.html
  2. Check that the page loads without JavaScript errors (open browser console)
  3. Verify that sample images load automatically
  4. Test navigation buttons (Previous/Next)
  5. Test keyboard shortcuts (Left/Right arrow keys)
  6. Verify that image counter updates correctly
  7. Check that navigation buttons are disabled at boundaries
  8. Verify that status banner shows sample mode notification

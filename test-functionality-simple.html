Functionality Test Instructions

This page tests the core functionality of the IMA Annotate Frontend:

  1. Load Sample Images: Click the "Load Sample Images" button to load images from the img folder
  2. Navigate Images: Use the Previous/Next buttons or arrow keys to navigate between images
  3. Draw Annotations: Click "Draw" mode and drag on the image to create bounding boxes
  4. Select Mode: Click "Select" mode to select and modify existing annotations
  5. ROI Mode: Click "ROI" to define regions of interest

Expected Behavior: The application should automatically fall back to sample mode since the API endpoints are dummy URLs.
